# 소프트웨어공학및실습 텀 프로젝트

## 개발

정보통신공학 유지훈(2016113519)

## 소개

- flask를 활용한 personal diary 개발 프로젝트  
- 나만의 diary 업로드 및 삭제
- 회원가입, 로그인, 로그아웃 기능 지원
- 이미지 업로드

## installation

본 프로젝트는 anaconda env 아래에서 개발되었습니다.

``` git clone https://github.com/sonagi784/SoftwareEngineering_hw.git ```  

SoftwareEngineering_hw 폴더 안에서,

``` pip install -r requirements.txt ```  

``` python app.py ```  

이후, 인터넷 브라우저를 통해 ```127.0.0.1:5000``` 에 접속해주세요.

## 환경

- flask  
- flask_sqlalchemy  
- anaconda
- sqlite
